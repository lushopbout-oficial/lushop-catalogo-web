# ⚡ GUÍA RÁPIDA - LU SHOP+ WEB

---

## 🚀 DEPLOYMENT EN 5 PASOS:

1. **GitHub:** Sube código → `lushop-catalogo-web`
2. **Apps Script:** Deploya API → Copia URL
3. **Vercel:** Conecta GitHub → Agrega variable de entorno
4. **GoDaddy:** Configura DNS (A + CNAME)
5. **¡Listo!** → https://lu-shop.com

---

## 📝 ACTUALIZAR PRODUCTOS:

1. Edita PRODUCTOS_MAESTRO en Google Sheets
2. ¡Ya! (automático, no hacer nada más)

---

## 📸 ACTUALIZAR FOTOS:

1. Sube foto a Drive
2. Obtén enlace compartido
3. Convierte a formato: `https://drive.google.com/uc?export=view&id=ID_AQUI`
4. Pega en columna M (Foto_URL)
5. ¡Ya! (automático)

---

## 🎨 CAMBIAR DISEÑO:

1. Edita archivos en GitHub
2. Commit changes
3. Vercel deploya automáticamente
4. Espera 1-2 min
5. ¡Cambios visibles!
